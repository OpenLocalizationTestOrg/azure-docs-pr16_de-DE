# <a name="how-to-change-profile-information-of-your-azure-account-such-as-contact-email-address-and-phone-number"></a>Zum Ändern der Profilinformationen Ihres Kontos Azure, wie Kontakt-e-Mail, Adresse und Telefon

## <a name="summary"></a>Zusammenfassung

Sie haben verpasst eine wichtige e-Mail von Azure? Möchten Sie weitere Kontaktinformationen in Ihrem Profil aktualisieren? In diesem Artikel lernen Sie:

-   [So ändern Sie Ihre Profilinformationen](#how-to-change-your-profile-information)
-   [Antworten auf häufige Probleme und Fragen](#frequently-asked-questions)

> [AZURE.NOTE] Wenn Sie an einer beliebigen Stelle in diesem Artikel weitere Hilfe benötigen, wenden Sie sich bitte [an den Support](https://portal.azure.com/?#blade/Microsoft_Azure_Support/HelpAndSupportBlade) , um das Problem erhalten schnell gelöst.

## <a name="important-information-about-your-profile-email-address"></a>Wichtige Informationen zu Ihrem Profil e-Mail-Adresse

Die e-Mail-Adresse in das Konto Administratorprofil wird von Microsoft verwendet, Sie wichtige Updates, die über das Abonnement die Abrechnung Zusammenhang informiert. Es empfiehlt sich, dass Sie eine Kontakt-e Mailadresse angeben, die Sie regelmäßig zu überprüfen.

## <a name="how-to-change-your-profile-information"></a>So ändern Sie Ihre Profilinformationen

1.  Melden Sie sich als Kontoadministrator [Azure Account Center](https://account.windowsazure.com/) , und klicken Sie dann auf **Profil**. Oder wechseln Sie zu https://account.windowsazure.com/Profile.

2.  Wählen Sie die Schaltfläche **Details bearbeiten** , und klicken Sie dann aktualisieren Sie **die Profilinformationen** .

## <a name="frequently-asked-questions"></a>Häufig gestellte Fragen

### <a name="can-i-change-the-account-administrator-email-address-in-my-profile"></a>Kann ich die e-Mail-Adresse Konto Administrator in Mein Profil ändern?

Ja. Konto-Administrator ist die Person, die die Azure-Konto eingerichtet und wer erhält wichtige e-Mail-Benachrichtigungen zu alle Abonnements unter dem Konto, an. Es empfiehlt sich, dass Sie eine Kontakt-e Mailadresse angeben, die dem Konto Administrator regelmäßig überprüft.

### <a name="does-updating-my-profile-email-also-update-my-login-email-address"></a>Aktualisieren e-Mail-Profil aktualisieren auch meine Login e-Mail-Adresse?

Nein. Aktualisieren die Profil e-Mail-Adresse aktualisiert Ihre e-Mail-Anmeldeadresse nicht. Wenn Sie Ihre e-Mail-Anmeldeadresse ändern möchten, müssen Sie den Besitz des Kontos zu übertragen.

### <a name="does-updating-my-profile-address-also-update-my-credit-card-billing-address"></a>Aktualisieren Mein Profil Adresse aktualisieren meiner Kreditkarte Rechnungsadresse auch?

Nein. Sie können die Rechnungsadresse Ihrer Kreditkarte zugeordnet sind, indem Sie auf der [Seite ' Abonnement '](https://account.windowsazure.com/subscriptions) in der Mitte der Azure-Konto aktualisieren.

### <a name="why-cant-i-update-the-country"></a>Warum werden kann nicht Land aktualisiert?

Aufgrund von technischen Einschränkungen können wir nicht Land für ein vorhandenes Konto ändern. Sie können jedoch erstellen ein neues Konto in das gewünschte Land und wenden Sie sich an Azure Support, um Ihre Services zu diesem Konto migrieren.

### <a name="what-email-address-does-the-azure-billing-alerts-preview-feature-use"></a>Welche e-Mail-Adresse der Azure Abrechnung Benachrichtigungen wird eine Vorschau der Features verwenden?

Wenn Sie für das neue Feature der Azure Abrechnung Benachrichtigungen Vorschau registrieren, erhalten Sie Azure Abrechnung Benachrichtigungen per e-Mail an die Adresse, die Sie angeben, e-Mail-Adresse, unabhängig davon, ob Sie der Administrator Konto geändert wurden.
