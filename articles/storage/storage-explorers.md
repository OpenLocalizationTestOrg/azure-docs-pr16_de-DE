# <a name="azure-storage-client-tools"></a>Clienttools Azure-Speicher

Häufig möchten Benutzer Azure-Speicher Ansicht/für ihre Daten mithilfe einer Azure-Speicher Clienttools interagieren können. In den folgenden Tabellen Listen wir eine Reihe von Tools, mit die Sie dies tun können. Wir setzen eine "X" in jedem Block aus, wenn es die Möglichkeit bietet, entweder auflisten und/oder die Datenabstraktion zugreifen. Die Tabelle enthält auch ist die Tools kostenlosen oder nicht. "Testen" gibt an, dass eine kostenlose Testversion vorhanden ist, aber die Vollversion nicht kostenlos ist. "Y/N" gibt an, dass eine Version zur Verfügung kostenlos, während eine andere Version kaufen verfügbar ist.

Wir haben nur eine Momentaufnahme der die verfügbaren Azure-Speicher Clienttools bereitgestellt. Diese Tools können weiterhin entwickeln und immer in Funktionalität. Wenn Sie Korrekturen oder Updates vorhanden sind, Schreiben Sie einen Kommentar zum lassen Sie uns wissen. Dies gilt auch, wenn Sie von Tools kennen, die hier - sein sollte wir zufrieden, um sie hinzuzufügen wäre.
